使用帮助

发表于 2013-09-11 | 分类于 blog

本博客使用markdown来写博客，下面主要介绍一些写博客的时候注意的东西

markdown指南

用markdown写作十分的方便，这里就主要给出几个链接吧：

  1. 鲁塔弗：markdown 简明语法
  2. 图灵社区：怎样使用Markdown
  3. 简书：献给写作者的 Markdown 新手指南
  4. 官方文档(中文版)：Markdown 语法说明

几种类型的文件的插入

  1. 嵌入图片

    图片的问题，可以采用博客新浪微博的图床，也可以放到自己的目录下面

    统一的格式为：![alt_text](url "可选的title")

    新浪微博图床的地址为：围博图床

    放入到自己的文件中：放到source目录下的pictures中，然后地址就写成/pictures/×.jpg

  2. 嵌入音乐

    以『虾米音乐』为例，歌曲页面有个『转帖』选项，将html代码或javascript代码复制到文中即可。

    示例代码：

    <embed src="http://www.xiami.com/widget/0_1333147/singlePlayer.swf" type="application/x-shockwave-flash" width="257" height="33" wmode="transparent"></embed>

    效果如下：

  3. 嵌入视频

    嵌入视频的方法和音乐类似，视频网站每个视频页面都会有一个『分享』或『转帖』按钮，点击可以查看代码。

    示例代码：

    <iframe height=498 width=510 src="http://player.youku.com/embed/XMjI2MjU3MDMy" frameborder=0 allowfullscreen></iframe>

    效果如下：

  4. 代码问题

    因为本人是一个程序员，所以贴代码什么的也就会经常出现了

    • 可以在“`”符号之间插入单行代码

    • 代码块，前后分别用三个“```”包围，然后在开头的反引号后面可以加上python等，表示代码高亮

几个注意的地方

  1. 首页隐藏

    在文章中的不需要在首页显示出来的部分之前添加: <!--more-->

  2. 考虑到博客标题是’h1’，所以博客内容的小标题以##开始

暂时先这么多吧。
